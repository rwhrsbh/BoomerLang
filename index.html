BoomerScript Guide

Welcome to BoomerScript, the programming language that'll make you say, "They don't code like they used to!"

Key Features:

  • backInMyDay: Function declaration (Because functions were better back in the day)
  • kidsTheseDays: Variable declaration (These young whippersnappers and their variables)
  • nowListenHere: Return statement (Pay attention, sonny!)
  • ifItAintBrokeDontFixIt: If statement (Why change what works?)
  • wellBackInMyDay: Else statement (Let me tell you how we did it...)
  • keepOnTruckin: While loop (Just keep going, like we did uphill both ways)
  • timesAreAChangin: For loop (But we'll adapt, by gum!)
  • tellEm: Print statement (Speak up, we can't hear you!)
  • whosYourDaddy: Equality check (Show some respect!)
  • oneMoreGo: Increment operator (One more for the road)
  • slowingDown: Decrement operator (Not as spry as we used to be)

Example:

backInMyDay addTwoNumbers(a, b) {
    kidsTheseDays result = a + b;
    nowListenHere result;
}

backInMyDay main() {
    kidsTheseDays x = 5;
    kidsTheseDays y = 10;
    kidsTheseDays sum = addTwoNumbers(x, y);
    
    ifItAintBrokeDontFixIt (sum whosYourDaddy 15) {
        tellEm("Sum is 15, just like I calculated on my abacus!");
    } wellBackInMyDay {
        tellEm("Sum ain't 15, must be new math!");
    }

    keepOnTruckin (x > 0) {
        tellEm("Counting down: " + x);
        slowingDown(x);
    }
}

main();
